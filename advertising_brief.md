# Рекламный Бриф: Кампания «ВЕКТОР: Направление на Рост»

## 1. Общая Информация

| Параметр | Значение |
| :--- | :--- |
| **Клиент** | Рекламное Агентство «ВЕКТОР» |
| **Продукт/Услуга** | Комплексные услуги по рекламе и PR для МСБ |
| **Период Кампании** | 3 месяца (Тестовый запуск) |
| **Цель Кампании** | Привлечение 10 новых клиентов из сегмента МСБ на комплексное обслуживание. |

## 2. Целевая Аудитория

*   **Основная ЦА:** Владельцы и руководители малого и среднего бизнеса (МСБ).
*   **География:** Крупные города, где сосредоточен активный МСБ.
*   **Проблемы ЦА:**
    *   Непрозрачность ценообразования и отчетности в маркетинге.
    *   Отсутствие четкого понимания, куда инвестировать рекламный бюджет.
    *   Страх "слить" бюджет впустую.
    *   Потребность в партнере, который говорит на их языке.

## 3. Ключевое Сообщение (Message)

### Главный Месседж: «Мы даем четкий вектор. Вы видите рост.»

**Поддерживающие сообщения:**
1.  **Прозрачность:** «Никаких скрытых комиссий. Только честная отчетность.»
2.  **Результат:** «Ваш бюджет — это инвестиция, а не расход. Покажем, как.»
3.  **Партнерство:** «Мы работаем *для* вашего бизнеса, а не *над* ним.»

## 4. Рекламное Оформление и Креативные Идеи

### 4.1. Визуальный Стиль
*   **Стиль:** Чистый, минималистичный, с акцентом на инфографику и цифры.
*   **Цвета:** Доминирующий Глубокий Синий, Акцентный Яркий Оранжевый для выделения ключевых цифр, стрелок и CTA.
*   **Изображения:** Использовать стилизованные изображения графиков роста, стрелок, направленных вверх, и рукопожатий (символ партнерства).

### 4.2. Креативные Идеи (Примеры Заголовков)
*   **Для контекстной рекламы (поиск):**
    *   "Реклама для МСБ: Четкий Вектор Роста"
    *   "Устали сливать бюджет? Прозрачный PR и Реклама."
    *   "Клиентоориентированное Агентство. Узнайте, как мы работаем."
*   **Для баннеров (визуальный акцент):**
    *   Баннер с крупной стрелкой (Оранжевый) и надписью: "Ваш бизнес здесь (Синий) → Ваш рост здесь (Оранжевый)".
    *   Баннер с графиком, где линия резко идет вверх, и подписью: "ВЕКТОР. Направление на результат."

## 5. Каналы Продвижения

1.  **Контекстная Реклама (Яндекс.Директ/Google Ads):** По запросам, связанным с "рекламное агентство для МСБ", "PR для малого бизнеса", "прозрачный маркетинг".
2.  **Социальные Сети (LinkedIn, Facebook):** Экспертный контент (кейсы, статьи о прозрачности бюджета) с таргетингом на руководителей.
3.  **Собственный Сайт/Лендинг:** Должен быть максимально информативным, с калькулятором ROI и четким описанием процесса работы (отражение прозрачности).

# Проект: Фирменный Стиль и Рекламное Оформление для Рекламного Агентства «ВЕКТОР»

## Обзор Проекта

Данный проект представляет собой разработку фирменного стиля и рекламного оформления для нового рекламного агентства, ориентированного на малый и средний бизнес (МСБ) с ключевым УТП — **клиентоориентированность**.

### Ключевые Элементы Бренда

| Элемент | Описание |
| :--- | :--- |
| **Название** | **«ВЕКТОР»** |
| **Слоган** | «Ваш вектор роста в рекламе и PR» |
| **Специализация** | Реклама и PR для МСБ |
| **Ключевая Ценность** | Клиентоориентированность и Прозрачность |

## Содержание Репозитория

| Файл | Описание |
| :--- | :--- |
| `creative_concept.md` | Разработка названия, слогана, легенды бренда и ключевых ценностей. |
| `visual_identity.md` | Описание концепции логотипа, цветовой палитры (Синий, Оранжевый) и типографики (Montserrat). |
| `brand_book_outline.md` | Структура и основные разделы Brand Book (миссия, видение, вербальная и визуальная идентичность). |
| `advertising_brief.md` | Рекламный бриф для первой кампании «ВЕКТОР: Направление на Рост», включая ЦА, ключевые сообщения и креативные идеи. |

---

## Креативная Концепция (Извлечение из `creative_concept.md`)

Название **«Вектор»** выбрано для отражения ключевого принципа работы агентства: **точное направление** и **фокусировка на цели клиента**. Оно ассоциируется с **движением**, **целенаправленностью** и **измеримым результатом**.

**Легенда бренда:**
> **«Вектор»** — это агентство, созданное для того, чтобы сделать эффективную рекламу и PR доступными и понятными для малого и среднего бизнеса. Мы строим свою работу на принципах **абсолютной клиентоориентированности**, что означает глубокое погружение в специфику бизнеса каждого клиента, разработку **индивидуальной стратегии** и **прозрачную отчетность**.

## Визуальная Идентичность (Извлечение из `visual_identity.md`)

**Цветовая Палитра:**
*   **Глубокий Синий** (`#004AAD`): Профессионализм, надежность.
*   **Яркий Оранжевый** (`#FF6600`): Энергия, рост, акцент.

**Типографика:**
*   **Основной Шрифт:** Montserrat (современный, чистый, легко читаемый).

## Рекламный Бриф (Извлечение из `advertising_brief.md`)

**Главный Месседж Кампании:** «Мы даем четкий вектор. Вы видите рост.»

**Креативная Идея:** Использование визуальных элементов (стрелок, графиков) и акцентного оранжевого цвета для выделения ключевых цифр и призывов к действию (CTA), подчеркивая **прозрачность** и **результативность**.
